# **App Name**: Academiq

## Core Features:

- Dashboard Overview: Display today's tasks, study sessions, and Pomodoro status with a motivational quote and AI-generated banner.
- Goal Tracking: Add semester and yearly goals, track progress with linked sub-tasks, and display completion percentages.
- Task Management: Add tasks with deadlines, priorities, and course tags; view tasks by 'Today', 'This Week', or 'All'.
- Course Management: Add courses with details, attach tasks/notes, and show a course progress indicator.
- Notes and Journal: Rich text editor for notes, tagged by course/project with a search bar. An AI tool will examine your existing notes to provide content-aware summarization to identify the main topic.
- Study Tracker: Log study sessions manually or auto-log Pomodoro sessions, and display weekly study hour charts.
- Pomodoro Timer: Customizable Pomodoro timer with focus mode and auto-logging to study tracker.
- Data Export/Import: Export and import app data as JSON files for backup and synchronization.
- Class Timetable Manager: User add courses and time to their timetable to easily track classes. Classes for the day show up in dashboard
- AI File Summarizer: User uploads file (PDF, pptx, txt), and the AI tool gives a concise summary. This summary can be saved to notes and linked to a course for easy navigation
- AI Flash Cards: User selects notes, and the AI tool generates flashcards from that note. Flashcards can be saved.
- AI Daily Planner: AI tool uses data about day from timetable, tasks etc., plus user's description of their desired day, to generate a very detailed hour-by-hour/minute-by-minute daily activities plan.
- AI Study Guide: User can generate study guide for their courses by uploading a scheme of work.
- More functionalities coming: More functionalities coming

## Style Guidelines:

- Primary color: Subtle Green (#8BC34A) to represent focus and growth. It will be used in different shades as backgrounds and accents to represent accomplishment and forward thinking. It is in the yellowish part of the green spectrum.
- Background color: Light Gray (#F5F5F5) to ensure a clean, distraction-free interface with good legibility, brightness 241, and very little saturation to work with dark text.
- Accent color: Soft Yellow (#DCE775) to highlight key actions and important information.
- Body and headline font: 'PT Sans', a humanist sans-serif for a modern yet approachable feel.
- Note: currently only Google Fonts are supported.
- Use clean, line-based icons from a consistent set to represent navigation items and actions.
- Employ rounded corners (2xl), soft shadows, and generous padding for a clean, modern card-based layout.
- Use subtle transitions powered by Framer Motion to provide smooth navigation and UI feedback.
